# 🎯 Jogo dos 7 Erros em Visualizações de Dados

Projeto desenvolvido com o objetivo de identificar, analisar e corrigir erros comuns em visualizações de dados, reforçando boas práticas de Data Visualization.

## 📌 Objetivo

Demonstrar, na prática, como más escolhas na construção de gráficos podem levar a interpretações equivocadas, além de apresentar versões corrigidas com melhorias visuais e analíticas.

## 🧠 Conceitos Abordados

- Escolha inadequada de tipos de gráficos
- Escalas inconsistentes ou enganosas
- Falta de normalização de dados
- Uso incorreto de cores
- Poluição visual (overplotting)
- Falta de contexto ou rótulos
- Comparações injustas entre métricas

## 🛠️ Tecnologias Utilizadas

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Estrutura do Projeto

```
├── Erros_em_Visualizacoes_de_Dados.ipynb
└── README.md
```

## 🔍 O que é realizado no Notebook

O notebook segue uma abordagem didática, apresentando:

1. Um gráfico com erro de visualização  
2. Explicação do problema  
3. Impacto na interpretação dos dados  
4. Versão corrigida do gráfico  
5. Justificativa da melhoria aplicada  

## 📊 Exemplos de Problemas Explorados

- Comparação de dados em escalas diferentes  
- Uso de gráficos inadequados para o tipo de dado  
- Eixos manipulados para distorcer interpretação  
- Excesso de informação visual  
- Falta de clareza nos rótulos  

## ✅ Resultados

O projeto permite entender como pequenas decisões na visualização podem impactar significativamente a interpretação dos dados, reforçando a importância de boas práticas em análise e comunicação de dados.
